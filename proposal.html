Project Topic: Convert Spotify Playlists into Bandcamp Links to Directly Support Artists

API Section

The RESTful APIs that will be used for this site are the Spotify API and the Bandcamp API.

The site will use the Spotify API to pull in the data from a playlist the user enters, then the Bandcamp API will be used to get the Bandcamp links for the songs in the playlists.

I want to also use the Spotify API to use the users' playlist to suggest other artists the user might enjoy, then return the Bandcamp links for those artists.

To sign up for the site, I'll ask for the users' first and last name, email, and date of birth. I'll also be collecting Spotify data from the user.

Possible DB tables and attributes:
  • USER -> user_id, first_name, last_name, dob, email, password, spotify_user_id
  • ARTIST -> artist_id, artist_spotify_id, artist_name, artist_bc_link
  • SONG -> song_id, song_spotify_id, song_name, song_bc_link

User Section

The people I see potentially using my site are people who want to directly support the artsits they enjoy on Spotify, since Bandcamp purchases go directly to the artist and streaming pays an average of $0.004 per stream.

Goals I have for my site are to promote directly supporting artists, and to make it easier to do so.
A large reason why people use streaming is convenience. I'm hoping my site will bring that element to being able to directly support the artist.

I chose this topic because music and supporting musicians is something that's important to me, specifically local artists who get the most screwed over by streaming.
